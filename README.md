# Wichtig

Wenn du das Projekt in Unity öffnest, geh zu:
**Assets > Scenes > MainScene**
Dort ist das ganze Level.

# Steuerung

* Bewegen: WASD
* Springen: Leertaste
* Sprinten: Shift
* Interagieren: E

# Features

* Lauf-, Sprint- und Jump-Animationen für den Mage
* Lever + Gate System
* Sound Effects und Background Music
* Particle Effects

Beim ersten Start in Unity kann es kurz dauern (lädt alles).
